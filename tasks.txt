# TASKS - LỊCH KHÁM SỨC KHỎE CÔNG TY HMSG CHC
## Danh sách công việc cần làm để phát triển và cải tiến hệ thống

### 📊 PHÂN TÍCH HIỆN TRẠNG
**Mô tả dự án:**
- Ứng dụng Google Apps Script quản lý lịch khám sức khỏe cho các công ty
- Dashboard hiển thị timeline khám theo tháng với giao diện hiện đại
- Tính năng: tìm kiếm công ty, filter nhân viên, ẩn/hiện công ty đã khám xong
- Tích hợp Google Sheets làm database
- Responsive design với UX/UI tối ưu

---

### 🚀 NHÓM 1: CẢI TIẾN UX/UI (Ưu tiên cao)

#### 1.1 Tối ưu hiệu suất và tốc độ
- [ ] **Implement Progressive Loading**: Tải dữ liệu theo chunk thay vì tải toàn bộ
- [ ] **Lazy Loading**: Chỉ render các dòng trong viewport
- [ ] **Virtual Scrolling**: Cho bảng timeline lớn (>100 công ty)
- [ ] **Optimize Cache Strategy**: Tăng thời gian cache, cache theo user
- [ ] **Minimize DOM manipulation**: Sử dụng DocumentFragment khi render

#### 1.2 Cải tiến giao diện người dùng
- [ ] **Dark Mode Toggle**: Thêm chế độ tối với color scheme riêng
- [ ] **Skeleton Loading**: Thay thế spinner bằng skeleton placeholder
- [ ] **Smooth Transitions**: Thêm animation cho navigation, filters
- [ ] **Sticky Header Improvements**: Cải thiện sticky behavior khi scroll
- [ ] **Mobile Navigation**: Redesign cho mobile với drawer navigation

#### 1.3 Accessibility & Usability
- [ ] **ARIA Labels**: Thêm accessibility attributes cho screen reader
- [ ] **Keyboard Navigation**: Tab navigation cho toàn bộ UI elements
- [ ] **Focus Management**: Quản lý focus khi mở/đóng modals
- [ ] **Color Contrast**: Đảm bảo WCAG 2.1 AA compliance
- [ ] **Text Scaling**: Support zoom 200% mà không bị break layout
